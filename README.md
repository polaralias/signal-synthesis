# signal-synthesis
A comprehensive trading companion app which utilises AI for identifying key stocks to watch and invest in.

## Android Project Structure
- Package name: `com.polaralias.signalsynthesis`
- App modules: `app` (single-module start)
- Layered packages:
  - `app/src/main/java/com/polaralias/signalsynthesis/data`
  - `app/src/main/java/com/polaralias/signalsynthesis/domain`
  - `app/src/main/java/com/polaralias/signalsynthesis/ui`

Notes: The folders above are placeholders for the Android project skeleton and
match the phased implementation plan.
